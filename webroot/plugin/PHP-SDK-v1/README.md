ACE php 离线 sdk Version 1.0
===================
ace php 的离线 sdk 用于线下开发, 兼容线上环境


sdk 配置
----------------------
请在 php.ini 配置:

    auto_prepend_file = [ace.php 的路径]


运行时文件目录 tmp
----------------------
位于 sdk 目录(ace.php所在的目录)的根目录下, 请确认 sdk 目录可写入

----------------------
