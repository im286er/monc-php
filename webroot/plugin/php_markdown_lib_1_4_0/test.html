wspyxz渠道特征分析

新增arpu = 可分配金额 / 新增用户数
    付费转化率 = 付费用户数 / 启动用户数
    次日留存率 = 次日留存用户数 / 新增用户数
    七日留存率 = 第七日留存用户数 / 新增用户数

总结

1）该渠道主推“单机”游戏，单机、网游的付费转化和留存均较差，总体用户质量差；

2）该渠道单款游戏的量均不大，最高只有900多/月,主要靠堆量；

3）单机“创意休闲”类的新增量较大，但是付费和留存较差；“体育竞技”类的付费转化率较高，量和留存较差，

 故：
    可以推荐渠道平台上表现不错的“创意休闲”“体育竞技”类游戏推广试下；

4）该渠道网游的付费转化和留存都不好，量也不大，不适合推网游；
